# Phase 0: Foundation

## Purpose
Project setup, configuration, and system prompts that define the core behavior of the assistant.

## Files in This Phase

### Configuration Files
- `requirements.txt` - All Python dependencies
- `.env` - Environment variables (API keys)
- `.streamlit/config.toml` - Streamlit theme (Groww colors)

### System Rules
- `system_prompt.md` - Facts-only rules and guardrails
- `config/sources.json` - List of Groww URLs to scrape

## Key Responsibilities
1. Define project dependencies
2. Configure API keys and environment
3. Set system rules and guardrails
4. List all data sources (Groww URLs)
5. Configure Streamlit theme
